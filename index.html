How This Report Works

✏️

Ticket Titles & Descriptions

Jira ticket summaries are sometimes rewritten here for clarity — making them more readable for a non-technical audience. The original wording is always one click away in Jira.

Filtering

Infrastructure tickets from Go migration epics (hs-openrtb → hs-apiary) are kept out of the Shipped/UAT sections to stay focused on product features. They appear in the Epic Progress sidebar instead.

🔍

Hover Previews

Hover any ticket row to see the full description. Click anywhere on the row to open it in Jira.

🔄

Auto-Refresh

Data is re-queried from Jira and this file is regenerated automatically each weekday at 8 AM.

Shipped — Recently Done

Issues moved to Done within the selected window. Toggle between last 10 days, 2 weeks, month, or quarter. The View filter in Jira ↗ link updates with the toggle.

📊

Epic Progress

Active epics with a completion bar (issues done / total). Use the All / SSP / Marketplace / Curate filter chips to focus on a specific product area. Hover any epic for a full description. Recently Wrapped Up shows epics fully closed in the last ~4 weeks.

📅

Epic Timeline (Gantt)

Click the 📅 Timeline button in the Epic Progress header to open a Gantt chart. Each bar spans the epic's planned start → due date, filled proportionally to completion. The dashed line marks today. Hover any row for the full epic card.
